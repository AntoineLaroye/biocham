créer une branche 
$ git branch nouvelle-branche

se déplacer 
$ git checkout nouvelle-branche

Pour contribuer on fork !

Puis on clone
$ git clone https://github.com/votre-username/intro-git-github.git

Nouvelle branche 
$ git checkout -b my-new-feature

Voire les différences
$ git diff

Ajouter les modiffications 
$ fit add .

Commiter
git commit -m "Added feature allowing users to comment on the blog articles"

pusher sur son repos 
git push origin my-new-feature

Puis pull request !
